# cyrboost

Очень простой скрипт, который собирает все кириллические сообщения в федиверс и репостит их.
Признаком сообщения на кириллице считается lang ru или 3 и больше кириллических символа.

Для запуска в mastodon клиентах понадобится создать приложение https://mastodon.social/settings/applications и использовать access token оттуда

Необходимые разешения для приложения:

  * read:statuses
  * profile
  * write:statuses
